# ユドナナシウム　ユドナリウムリリィの避難所

[ユドナリウム（Udonarium）][udonarium-url]を改造して作った、新SkyWay対応のユドナリウムリリィの避難所です。
ユドナリウムリリィが新SkyWayに対応して完全復活を遂げたらお払い箱になります。

## 機能

- **移植できてない機能**
　- ターゲット機能
　- マップマスクのスクラッチ
　- ログ消去機能
　- 点呼＆投票
　- アラーム
  - コマのコピー時にナンバーを振る
　- 射程距離
　- 全員グリッド強制表示＆デフォルトダイスボット指定
　- カウンターリモコン
　- インベントリの第二ソート条件
　- 「単一チェック」「チェック/表」の選択肢
　- 接続秒数確認
  - その他、なんか見落としてるかも
  - ユドナリウムにあったコマ一括操作機能（いじってるうちに消えちゃった）

- **既知の不具合**
  - 立ち絵差分登録時に出てくるウィンドウは、自動的に消えません。消す以外の操作も受け付けません。
    けどまあ、最初の登録がだるいだけでプレイには支障ないと思います。
